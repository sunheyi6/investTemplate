## 🎯 核心投资哲学

### 一级市场思维
> **"10元买瓜，自己吃也值15元"**
> - 买入价 < 内在价值
> - 保底收益 > 7%
> - 无需看盘，睡觉赚钱

### 烟蒂股思维
> **"赚价值修复的钱，不赚成长的钱"**
> - 从极端低估修复到合理估值就离开
> - 下行边际要强（净现金 > 市值×2倍）
> - 三种策略：纯硬收息型 / 正股增强型 / 清算套利型
