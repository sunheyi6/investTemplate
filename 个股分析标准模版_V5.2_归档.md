## 第一章：流派识别与分流

### 1.1 初筛分流矩阵（关键第一步）

面对任何标的，首先用以下标准判断所属流派：

| 筛选维度 | 🛡️ 纯硬收息型 | 🔍 价值发现型 | 🚬 烟蒂股 | 🔗 关联方资源型（新增） |
|---------|-------------|-------------|-----------|---------------------|
| **核心诉求** | 股息现金流 | 估值修复+成长 | 清算价值套利 | 母公司资源赋能+轻资产现金流 |
| **股息率门槛** | **>7%** | 3-6%（可接受） | 0-3%（不关注） | 3-5%（稳定增长型） |
| **PE区间** | <8倍 | 8-18倍（分类型） | 亏损或PE<5 | **12-18倍**（可放宽） |
| **PB区间** | <0.6倍 | 0.6-1.5倍 | **<0.3倍** | 1.5-3倍（轻资产高ROE） |
| **净现金要求** | >市值30% | 有即可 | **>市值×2倍** | 轻资产，净现金为正即可 |
| **核心资产** | 稳定现金流资产 | FCF/品牌/控股折价 | 清算现金 | **母公司资源合同** |
| **业务特征** | 简单稳定（银行/能源） | 轻资产/控股平台 | 重资产/困境 | **轻资产+强关联母公司** |
| **持有期限** | 3-5年起 | 1-3年（修复即走） | 6-18个月（事件驱动） | 3-5年（跟随母公司成长） |
| **典型标的** | 建行H、中石化 | 蒙牛、华润医药 | 天津发展、庆铃 | **中海物业、华润万象** |

#### 特殊识别规则：

**规则1：多标准冲突时的优先级**
```
if 净现金>市值×2倍 and PB<0.3:
    优先归类 = 🚬 烟蒂股（风险收益比最高）
elif 股息率>7% and 连续5年分红:
    优先归类 = 🛡️ 纯收息（保底收益优先）
elif 轻资产 and 母公司为行业龙头 and PE<15:
    优先归类 = 🔗 关联方资源型（特殊价值）
elif EV/FCF<12倍 or 控股折价>30%:
    优先归类 = 🔍 价值发现（FCEV或控股平台）
else:
    进入观察区/排除
```

**规则2：关联方资源型的快速识别**
- 标的为**轻资产服务业**（物管、商管、供应链、代建）
- **母公司为地产/基建/制造龙头**（央国企优先）
- **关联交易占比>30%** but **定价公允**
- **PE介于12-18倍**（高于纯烟蒂，低于成长股）

### 1.2 排除清单（全流派通用）

| 排除项 | 说明 | 例外情况 |
|--------|------|---------|
| ❌ 市值<5亿港元 | 流动性风险 | 烟蒂股可放宽至3亿，仓位<2% |
| ❌ 日均成交<100万 | 无法买卖 | 纯收息/关联方资源型可放宽 |
| ❌ 有息负债>净现金×3倍 | 财务风险 | 困境反转型（地产）需个案评估 |
| ❌ 连续3年不分红 | 治理风险 | 烟蒂股允许（但有息负债必须极低） |
| ❌ 大股东质押率>50% | 治理风险 | 无例外 |
| ❌ 人民币收入<50%且股息率<8% | 汇率风险 | 纯收息型需特别注意 |
| ❌ 关联交易定价不公允 | 利益输送风险 | **关联方资源型一票否决** |

### 1.3 假性高估识别（反向筛选）

独立于四流派体系，识别"表面贵实际便宜"的机会：

| 类型 | 识别标准 | 验证方法 | 典型标的 |
|------|---------|---------|---------|
| ① 非现金减值 | 投资物业减值、商誉减值 | 经营现金流>>净利润 | 香港REITs、长和 |
| ② 一次性支出 | 补税、罚款、诉讼和解 | 第二年自动消失 | 福寿园（历史案例） |
| ③ 会计重组 | 并表转权益法 | 股权重估折扣 | 长和（英国三集团剥离） |
| ④ 汇兑损失 | 本币升值账面损失 | 不影响实际经营 | 人民币资产港元计价 |

**判断标准（必须同时满足）**：
1. 非现金或一次性（不影响实际现金流）
2. 短期不持续（不会每年都发生）
3. 存量资产不受损（资产负债表健康）
4. 即使按真实利润估值仍有折扣

## 第二章：纯硬收息型分析框架 🛡️

> **核心哲学**："每年7-8%股息到账，股价波动与我无关"

### 2.1 收息专用数据清洗

| 检查项 | 标准 | 权重 | 说明 |
|--------|------|------|------|
| **A1_TTM** | 最近4季度扣非净利 | 20% | 用于计算派息覆盖率 |
| **B1分红记录** | 连续5年+分红 | 30% | 必须连续，中断直接排除 |
| **B1派息率** | 30-70%区间 | 20% | <30%吝啬，>70%不可持续 |
| **NetCash** | >市值30% | 15% | 财务安全垫 |
| **利率敏感度** | 降息周期受益 | 15% | 高负债企业（如银行）受益 |

### 2.2 V2.0高股息估值法（债权人视角）

**核心公式**：
```
收息买点 = (A1_TTM × B1_派息率 × 0.8 + C1_回购) / 目标股息率 ÷ 总股本

其中：
- 目标股息率 = 7%（税前）/ 5.6%（税后，考虑20%红利税）
- 0.8 = 安全边际（利润下降20%仍能维持股息）
- C1 = 仅计入注销式回购金额
```

**简化判断标准**：
| 指标 | 买入标准 | 持有标准 | 卖出标准 |
|------|---------|---------|---------|
| **股息率** | >7% | 5-7% | <5% |
| **派息覆盖率** | <60% | 60-80% | >100%（举债派息） |
| **股价vs买点** | 低于买点 | 持平 | 高于买点30%+ |

### 2.3 收息型行业适配度

| 行业 | 适配度 | 代表标的 | 注意点 |
|------|-------|---------|-------|
| **银行** | ⭐⭐⭐⭐⭐ | 建行H、工行H | 关注净息差走势 |
| **能源（油/煤）** | ⭐⭐⭐⭐ | 中石化、中国神华 | 周期性，需派息承诺 |
| **公用事业** | ⭐⭐⭐⭐⭐ | 华润电力、中电控股 | 稳定但增长有限 |
| **电信** | ⭐⭐⭐⭐ | 中国移动、中国电信 | 5G资本开支下降后改善 |
| **航运港口** | ⭐⭐⭐ | 中远海控、东方海外 | 周期性强，高股息时往往见顶 |
| **REITs** | ⭐⭐ | 领展房产基金 | 管理费抽血风险（参考汇贤） |

### 2.4 收息型持仓管理

**操作纪律**：
- ✅ **买入后**：不看行情软件，每年只看4次（季报发布日）
- ✅ **股息到账**：可再投入或提取作为现金流
- ✅ **年度体检**：检查派息率是否>100%、负债是否恶化
- ❌ **禁止操作**：股价涨了20%就卖出（违背收息初心）

**退出条件（触发任一即卖出）**：
1. 股息率<5%（估值修复过度）
2. 连续2年削减派息
3. 派息率>100%（不可持续）
4. 发现更好标的（股息率>当前+2%）

## 第三章：价值发现型分析框架 🔍

> **核心哲学**："剔除净现金后，用自由现金流收益率衡量真实价值；或通过控股折价/母公司资源获取超额收益"

### 3.1 流派细分：四大子类

| 维度 | A. FCEV轻资产型 | B. 品牌护城河型 | C. 困境反转型 | D. 控股平台型 | **E. 关联方资源型（新增）** |
|------|----------------|----------------|--------------|--------------|---------------------|
| **核心资产** | 品牌/渠道/轻资产模式 | 品牌溢价+同店增长 | 周期底部+存活能力 | 持有上市公司股权 | **母公司资源合同** |
| **估值锚点** | EV/FCF < 10倍 | PE 12-18倍+ROE>15% | PB<0.5倍+现金流为正 | 控股折价率 > 30% | **PE<15倍+母公司质地** |
| **FCF特征** | FCF ≈ 净利润 | FCF好但可容忍扩张期 | FCF暂时为负/不稳定 | 子公司分红穿透 | **轻资产预收模式** |
| **典型标的** | 蒙牛、同仁堂国药 | 海底捞、颐海国际 | 中海地产、华润置地 | 华润医药、海螺创业 | **中海物业、华润万象** |
| **退出信号** | EV/FCF > 15倍 | PE>25倍或品牌老化 | 行业复苏PE>12倍 | 折价率 < 10% | **母公司资源枯竭** |

### 3.2 A. V3.5 FCEV估值法（轻资产视角）

**适用条件**：
- ✅ 净现金极高（资产负债率<20%）
- ✅ 轻资产模式（资本开支/净利润 < 20%）
- ✅ FCF稳定且≈净利润

**核心公式**：
```
EV = 市值 - NetCash（扣除有息负债后的净现金）
FCF = 经营现金流净额 - 资本开支
EV/FCF = EV / FCF
FCF收益率 = FCF / EV × 100%
```

**估值对照表**：
| EV/FCF区间 | FCF收益率 | 评级 | 操作建议 |
|-----------|----------|------|---------|
| <8倍 | >12.5% | 🐢🍊 **金龟** | 重仓买入 |
| 8-10倍 | 10-12.5% | 🐢💎 **银龟** | 分批建仓 |
| 10-12倍 | 8.3-10% | 🐢💰 **铜龟** | 持有/观望 |
| 12-16倍 | 6.25-8.3% | ⚠️ 合理偏高 | 不买入 |
| >16倍 | <6.25% | ❌ 不符合 | 等待回调 |

### 3.3 B. 品牌护城河型（新增子类）

**适用标的**：餐饮连锁、消费品、具备强品牌溢价的轻资产公司

**与FCEV型的区别**：
- FCEV型看**EV/FCF**（<10倍）
- 品牌型看**PE+FCF收益率+品牌强度**（允许PE 12-18倍）

**筛选标准**：
| 检查项 | 标准 | 权重 | 说明 |
|--------|------|------|------|
| **品牌强度** | 行业TOP3或细分龙头 | 30% | 海底捞、九毛九、颐海国际 |
| **ROE** | >15%且持续3年 | 20% | 品牌溢价转化为股东回报 |
| **同店增长** | 正增长或企稳 | 20% | 品牌未老化 |
| **FCF/市值** | >5% | 20% | 自由现金流收益率底线 |
| **PE** | <18倍（较历史中枢折价） | 10% | 估值安全边际 |

**买卖点规则**：
```python
if PE < 15倍 and 同店增长 >= 0:
    评级 = 🐢🍊 金龟
    买入
elif PE < 18倍 and 较历史高点回撤 > 30%:
    评级 = 🐢💎 银龟
    分批买入
elif PE > 25倍 or 同店增长连续2年负增长:
    卖出（品牌溢价泡沫化或老化）
```

### 3.4 C. 困境反转型（新增子类）

**适用标的**：地产开发商、周期股底部、FCF暂时不稳定但存活能力强的央企

**与烟蒂股的区别**：
- 烟蒂股：净现金>市值×2倍，有息负债接近零，可清算
- 困境反转：**有息负债高（行业特性）**，但**经营现金流为正**，依赖央企信用

**筛选标准（以地产为例）**：
| 检查项 | 合格标准 | 一票否决 |
|--------|---------|---------|
| **央企/国企背景** | 必须 | 民企地产（当前环境） |
| **经营现金流** | 连续2年为正 | 经营现金流为负（失去造血能力） |
| **有息负债/经营现金流** | <3倍 | >5倍（偿债困难） |
| **土储质量** | 一二线城市占比>60% | 三四线为主（去化困难） |
| **PB** | <0.5倍 | >0.8倍（安全边际不足） |
| **PE** | <10倍 | >15倍（非困境状态） |

**仓位限制**：
- 单只困境反转型标的 **≤5%**（高不确定性）
- 困境反转总仓位 **≤15%**
- 必须分散3只以上（不同行业）

### 3.5 D. 控股平台估值法（正股增强）

**适用标的**：母公司为控股平台，核心资产为上市子公司股权

**分析步骤**：

**步骤1：计算持股市值**
```
持股市值 = Σ(各子公司市值 × 母公司持股比例)
```

**步骤2：计算控股折价率**
```
控股折价率 = (持股市值 - 母公司市值) / 持股市值 × 100%
```

**步骤3：综合价值评估**
```
母公司真实价值 = 持股市值 + 母公司自身业务估值 + 净现金 - 治理折价

其中：
- 自身业务估值：若有独立业务，按PE 8-12倍计算
- 治理折价：10-30%（视关联交易、管理层诚信而定）
```

**买入标准**：
| 检查项 | 标准 | 权重 |
|--------|------|------|
| 控股折价率 | >30% | 30% |
| 目标公司质地 | 行业龙头，PE<15 | 25% |
| 母公司自身价值 | 有稳定业务或净现金 | 20% |
| 催化剂 | 目标公司估值修复预期 | 15% |
| 流动性 | 日均成交额>100万港元 | 10% |

### 3.6 E. 关联方资源型（V5.2新增核心子类）

**定义**：轻资产子公司，核心业务增长高度依赖优质母公司的资源输送，但本身具备独立造血能力和定价权。

**典型标的**：
- **物管公司**：中海物业（依赖中海地产）、华润万象生活（依赖华润置地）
- **供应链公司**：绿城管理（代建）、建筑类供应链
- **商管公司**：宝龙商业、新城悦服务

**与控股平台型的本质区别**：
| 维度 | 控股平台型（如华润医药） | 关联方资源型（如中海物业） |
|------|------------------------|--------------------------|
| **股权关系** | 母公司持有子公司 | 子公司被母公司持有 |
| **价值来源** | 子公司市值折价 | 母公司资源输送合同 |
| **估值锚点** | 控股折价率 | PE+合同增长预期 |
| **风险点** | 折价永不收敛 | 母公司资源枯竭或定价不公允 |

**专用筛选标准（六维模型）**：

| 维度 | 合格标准 | 中海物业验证 | 权重 |
|------|---------|-------------|------|
| **① 母公司质地** | 行业龙头，财务稳健（央国企优先） | ✅ 中海地产，央企，高端地产第二 | 25% |
| **② 资源可持续性** | 母公司土储/项目储备>5年输送能力 | ✅ 中海地产土储充足，且聚焦高端 | 20% |
| **③ 定价公允性** | 关联交易价格 vs 第三方价格差异<20% | ⚠️ 需查财报（物管费通常公允） | 20% |
| **④ 独立性验证** | 第三方外拓占比>20%（防过度依赖） | ⚠️ 需跟踪（理想>30%） | 15% |
| **⑤ 估值安全边际** | PE<15倍 或 较历史PE中枢折价30% | ✅ 当前若12-15倍PE可接受 | 15% |
| **⑥ FCF质量** | 经营现金流/净利润>80%（预收模式） | ✅ 物管行业先收费后服务 | 5% |

**估值与买卖点规则**：

**估值逻辑**：
```
合理PE = 基础PE（10倍） + 母公司质量溢价（0-5倍）

其中：
- 基础PE：轻资产服务业基准
- 母公司质量溢价：
  - 央企龙头+土储优质：+3-5倍PE
  - 国企中等：+1-2倍PE
  - 民企或土储一般：+0倍PE（甚至折价）
```

**买卖标准**：
| 情景 | PE区间 | 评级 | 操作 |
|------|-------|------|------|
| 悲观情景 | <12倍 | 🐢🍊 **金龟** | 重仓买入（资源+估值双保险） |
| 合理情景 | 12-15倍 | 🐢💎 **银龟** | 分批买入（资源溢价已部分反映） |
| 乐观情景 | 15-20倍 | 🐢💰 **铜龟** | 持有/不买入 |
| 泡沫情景 | >20倍 | ❌ **卖出** | 资源溢价过度透支 |

**特殊退出条件（区别于其他流派）**：
```python
if 母公司出现财务危机（如债务违约、现金流断裂）:
    立即卖出（资源逻辑破坏，无论估值多低）
elif 关联交易定价明显恶化（如物管费大幅低于市场价）:
    立即卖出（利益输送风险）
elif 第三方外拓停滞（过度依赖母公司，独立性丧失）:
    降级/减仓（增长天花板显现）
elif PE > 20倍:
    止盈卖出（估值透支）
```

**与母公司股票的配对交易**：
- **一鱼两吃策略**：同时持有中海地产（困境反转）+ 中海物业（关联方资源）
- **对冲逻辑**：若地产复苏，物业也有增量；若地产继续低迷，物业现金流仍稳
- **仓位配比**：地产:物业 = 1:2（物业轻资产风险更低，可给更高仓位）

### 3.7 价值发现型持仓管理

**与纯收息型的关键区别**：
- **跟踪频率**：每季度（而非每年），关注FCF/折价率/母公司资源变化
- **卖出纪律**：估值修复到位即卖出（不恋战），不同于收息型长期持有

**卖出标准**：
- FCEV型：EV/FCF >15倍或相对行业龙头折价<10%
- 品牌型：PE>25倍或同店增长连续为负
- 困境型：PB>0.8倍或行业复苏信号明确（见好就收）
- 控股平台型：折价率<10%或子公司估值泡沫化
- **关联方资源型：母公司资源枯竭或PE>20倍**

**再投资规则**：
- 卖出后优先寻找同流派标的（保持风格纯粹）
- 若市场无合适标的，暂时转入纯收息型作为过渡

#### 4.2.1 六维兑现概率评估模型（V5.3新增）

> **核心洞察**：PB 0.2不是买入理由，"PB 0.2 + 高兑现概率"才是

| 评估维度 | 权重 | 高概率特征（+分） | 低概率特征（-分） | 数据来源 |
|---------|------|------------------|------------------|---------|
| **① 大股东动机** | 25% | 持股>60%有私有化动力、央企背景、股价长期低迷倒逼改革 | 持股<30%、民企、管理层佛系、无市值管理压力 | 年报/披露易 |
| **② 监管环境** | 20% | 国企改革政策推动、提高股东回报要求、监管点名 | 监管宽松、无政策压力、行业不受关注 | 国务院/国资委文件 |
| **③ 财务能力** | 20% | 净现金>市值×3倍、无大额资本开支、FCF充裕 | 现金紧张、有并购计划、烧钱速度快 | 财报/现金流量表 |
| **④ 历史行为** | 15% | 历史上曾回购/私有化/特别派息、分红稳定 | 从未有兑现行为、分红随意、承诺不兑现 | 历史公告 |
| **⑤ 流动性压力** | 10% | 日均成交<100万、机构持仓少、融资功能丧失 | 流动性充裕、机构关注、融资能力强 | 港交所数据 |
| **⑥ 行业周期** | 10% | 行业底部、资产处置窗口期、政策催化 | 行业高峰、资产价格泡沫、转型期 | 行业研报 |

**评分标准**：
```
兑现概率 = Σ(各维度得分 × 权重)

- 80-100%：极高概率，可重仓（8%仓位）
- 60-80%：高概率，标准仓位（5%仓位）
- 40-60%：中等概率，观望或小仓位（2%仓位）
- <40%：低概率，排除（避免价值陷阱）
```

**典型案例对比**：
- ✅ 庆铃汽车：兑现概率85%（新能源催化+现金充裕+国资委压力）→ 5倍涨幅
- ⚠️ 北京控股环境：兑现概率60%（2024年零分红，动机不明）→ 观望
- ❌ 某民企烟蒂：兑现概率30%（管理层佛系，无政策压力）→ 排除（0.2PB可杀至0.1PB）|

### 4.3 烟蒂股专用估值法

**极端保守估值**：
```
烟蒂买点 = 净现金 × 0.8 + (年均股息 × 10年)

或更简化：
烟蒂买点 = 市值 × 0.5（当净现金 > 市值时）
```

**卖出目标位**：
```
目标市值 = 净现金 × 1.0 或 PB修复至0.5倍（取较低者）

预期收益率 = (目标市值 - 当前市值) / 当前市值 × 100%
通常目标：50-100%收益，修复即离开
```

#### 4.5.3 估值重塑目标价计算（V5.3改进版）

**情景A：私有化/回购兑现（概率>80%）**
```
目标价 = 净现金 × 0.9
预期涨幅 = 3-5倍
仓位建议：8%（重仓）
```

**情景B：恢复派息（概率60-80%）**
```
目标价 = (年均利润 × 派息率) / 目标股息率(6%)
预期涨幅 = 2-3倍（股息率从0%→6%，PB从0.2→0.5）
仓位建议：5%（标准）
```

**情景C：部分兑现（回购+派息，概率40-60%）**
```
目标价 = 净现金 × 0.6 + (年均利润 × 派息率) / 8%
预期涨幅 = 1-2倍
仓位建议：2%（小仓位）
```

**情景D：无兑现（概率<40%，概率止损）**
```
卖出价 = 净现金 × 0.5（安全边际耗尽）
预期亏损 = -20%至-50%（0.2PB→0.1PB）
仓位建议：0%（排除）
```

- 烟蒂股总仓位 ≤ 20%
- 必须分散3只以上（非系统性风险）

```
    ↓ 否 → 谨慎/降低仓位
```

### 6.2 🔍 价值发现型决策流程（含四子类）

```

识别标的类型：
```

### 7.2 流派间再平衡规则

**定期再平衡（年度）**：
- 检查各流派仓位是否偏离目标±10%

**事件驱动再平衡**：

### 7.4 流派转换规则

**允许转换的情形**：
1. **烟蒂股 → 纯收息**：烟蒂股修复后若股息率>7%，可转为收息持仓
2. **价值发现 → 纯收息**：估值修复完成后若估值仍低且股息高，可过渡
3. **关联方资源型 → 纯收息**：若母公司资源枯竭但标的自身股息率>7%，可降级为收息型

**禁止转换的情形**：
- 关联方资源型母公司财务危机 → 必须卖出，**不可**转为其他流派（资源逻辑已破坏）
- 价值发现标的估值泡沫化 → 必须卖出，不可转为收息（股息率不足）

- **目标价计算**：按兑现概率分四档（私有化/派息恢复/部分兑现/无兑现）

**V5.2 完整修订版** (2026-02-24)
- **重大更新**：增加"关联方资源型（母公司赋能型）"作为第四大流派
- **新增子类**：价值发现型扩展为FCEV/品牌/困境/控股平台/关联方资源 五类
- **解决盲区**：中海物业等轻资产高PE but 强母公司赋能标的归类问题
- **新增案例**：中海物业、华润万象生活完整分析
- **风险提示**：关联方资源型特有的母公司财务风险监控指标

**适用说明**：
本模板为个人投资研究使用，不构成投资建议。使用者应根据自身风险承受能力选择合适流派，并严格执行各流派的买入/卖出纪律。

**特别警告（烟蒂股-V5.3新增）**：
PB 0.2不是买入理由，"PB 0.2 + 高兑现概率"才是。没有兑现逻辑的烟蒂股，0.2PB可以杀到0.1PB。必须使用六维模型评估兑现概率，<40%直接排除。

**特别警告（关联方资源型）**：
该流派核心逻辑依赖母公司资源输送，**一旦母公司出现财务危机或关联交易定价恶化，必须立即卖出**，无论估值多低。这是与纯收息型、FCEV型最大的区别。
