## 第四章：估值分析

### 4.1 市值计算与交叉验证

```
市值 = 股价 × 总股本
     = 23.96港元 × 2.764亿股
     = 66.23亿港元
     = 61.39亿人民币（汇率0.927）

交叉验证：
├── 腾讯API显示：66.23亿港元 ✅
└── 理杏仁数据：约71.73亿（基于25.95元股价，已过时）
```

### 4.2 PE估值（静态+TTM双口径）

```
静态PE(2024) = 市值 / 2024年净利润
             = 66.23 / (5.47/0.927)
             = 66.23 / 5.90
             = 11.22倍

TTM净利润 = 2025H1(2.98亿) + 2024H2(2.81亿)
          = 5.79亿人民币
          = 6.25亿港元

TTM PE = 66.23 / 6.25 = 10.60倍
```

**PE交叉验证**：
| 来源 | PE | 差异原因 |
|------|-----|---------|
| 手动计算（静态） | 11.22倍 | 基于2024全年 |
| 手动计算（TTM） | 10.60倍 | 基于最新12个月 |
| 腾讯API | 10.04倍 | 可能使用调整后EPS |

### 4.3 剔除净现金PE（核心指标）

```
净现金 = 32亿人民币 = 34.52亿港元

剔除净现金PE = (市值 - 净现金) / 净利润
             = (66.23 - 34.52) / 5.90
             = 31.71 / 5.90
```
