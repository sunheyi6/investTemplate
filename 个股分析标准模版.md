# 知识库协议更新：个股分析标准模版(V4.1)

**适用于所有类型个股（成长/价值/周期）的深度体检**

---

## 0. 金龟筛选框架(筛选前置)

### 0.1 初筛条件（满足任一即进入深度分析）

| 筛选维度 | 金龟标准 | 银龟标准 | 说明 |
|----------|----------|----------|------|
| **估值水平** | PB < 0.8 或 PE < 8 | PB < 1.2 或 PE < 12 | 破净或低估值 |
| **股息率** | > 6% | > 4% | V2.0收息视角 |
| **净现金** | 净现金 > 市值×30% | 净现金 > 0 | 财务安全垫 |
| **EV/FCF** | < 10倍 | < 15倍 | V3.5自由现金流视角 |
| **市值折价** | 市值 < V3.0估值×50% | 市值 < V3.0估值×70% | 极度低估 |

### 0.2 行业特征标签（优先关注）

**高优先级标签：**
- ✅ **垄断/牌照壁垒**：GDS、交易所、公用事业、牌照稀缺行业
- ✅ **央国企背景**：国资委控股、地方国企、分红有保障
- ✅ **轻资产高现金**：FCF ≈ 净利润、资本开支/净利润 < 20%
- ✅ **母子公司架构**：控股平台、核心资产可穿透计算

**中优先级标签：**
- ⚠️ 周期性底部、行业拐点、政策驱动

### 0.3 排除清单（直接过滤）

| 排除项 | 说明 |
|--------|------|
| ❌ 市值 < 5亿港元 | 流动性极差，无法配置 |
| ❌ 日均成交额 < 100万港元 | 流动性不足，难以买卖 |
| ❌ 有息负债 > 净现金×3倍 | 财务风险过高 |
| ❌ 连续3年亏损或不分红 | 基本面恶化 |
| ❌ 大股东质押率 > 50% | 治理风险 |

> **注**：市值5-10亿港元的标的，若符合金龟标准（净现金>市值、高股息等），可个案考虑，但需降低仓位上限。

## 1. 核心数据清洗(Data Cleaning)

一切估值的前提。拒绝盲目引用软件数据，必须手动穿透。

### 1.1 真实盈利(A1)

| 版本 | 计算方法 | 适用场景 |
|------|----------|----------|
| **TTM版** | 最近4个季度扣非净利 | 用于V2.0高股息估值 |
| **悲观版** | 若遇行业拐点，强制年化最新单季低点 | 用于V3.0保底估值 |

### 1.2 分红意愿(B1)

- 承诺支付率 vs 实际支付率
- 连续分红年限
- 派息率稳定性

### 1.3 回购含金量(C1)

- 仅计入注销式回购金额
- 回购/市值比例

### 1.4 败金金额(Mal)

非主业亏损 + 低效资本开支（V3.0专用扣减项）

### 1.5 安全气囊(NetCash)

```
NetCash = (货币资金 - 有息负债) × 折扣系数
```

折扣系数参考：
- 现金及等价物：1.0
- 受限资金：0.5
- 应收账款：0.3

### 1.6 穿透回报(D1) ⭐

分析核心子公司对母公司的利润/分红穿透贡献

| 检查项 | 验证内容 | 备注 |
|--------|----------|------|
| 持股比例 | 母公司对核心子公司的实际持股比例 | 并表/权益法核算 |
| 利润穿透 | 子公司净利润 × 持股比例 | 核心利润来源识别 |
| 分红穿透 | 子公司分红 × 持股比例 | 现金流回母公司的能力 |
| 控制权分析 | 是否有实际控制权 | 少数股东权益摊薄评估 |

## 2. 左侧轨道: V2.0高股息协议(The Bond View)

- **适用视角**: 债权人/收息族
- **核心逻辑**: 相信公司的永续经营和分红承诺

### 计算公式

```
PR = N × (PE/ROE)

收息买点 = (A1 × B1 × 0.8 + C1) / 4% ÷ 总股本
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 0.8 | 安全边际系数 | 保守估计 |
| 4% | 目标股息率 | 无风险利率+风险溢价 |

---

## 3. 右侧轨道: V3.0 10倍估值法(The Owner View)

- **适用视角**: 私有化财团/深度价投
- **核心逻辑**: 假设没有增长，仅靠存量利润10年回本+白送现金

### 计算公式

```
正常市值 = (A1_TTM × 0.8 × 10) + NetCash

保底市值 = [(A1_悲观 × 0.8) - Mal] × 10 + NetCash
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 0.8 | 利润可持续性折扣 | 保守估计 |
| 10 | 10年回本倍数 | 无增长假设 |
| Mal | 败金金额扣减 | V3.0专用 |

## 4. 中轴轨道: V3.5 自由现金流企业价值法(The FCEV View) ⭐

- **适用视角**: 轻资产高现金流的消费/品牌企业
- **核心逻辑**: 剔除净现金后，用自由现金流收益率对标行业龙头估值
- **适用条件**: NetCash极高、轻资产、FCF≈净利润、PE/PB看似偏高但EV/FCF极低

### 计算公式

```
EV = 市值 - NetCash
FCF = 经营现金流净额 - 资本开支
FCF收益率 = FCF / EV
EV/FCF = EV / FCF
```

### 对比基准

| 对标类型 | FCF收益率 | EV/FCF | 适用场景 |
|----------|-----------|--------|----------|
| 消费龙头(肯德基/海底捞) | 5-6% | 16-20倍 | 品牌护城河+稳定现金流 |
| 价值买点 | >8% | <12倍 | 显著折价 |
| 极度低估 | >10% | <10倍 | 无脑买入区间 |

### 评估标准

| EV/FCF区间 | 相对消费龙头折价 | 评级 | 操作建议 |
|------------|------------------|------|----------|
| <10倍 | >50%折价 | 🐢🍊 **金龟** | 无脑买入 |
| 10-12倍 | 30-50%折价 | 🐢💎 **银龟** | 分批建仓 |
| 12-16倍 | 0-30%折价 | 🐢💰 **铜龟** | 观望/持有 |
| >16倍 | 无折价 | ❌ **不符合** | 等待回调 |

### 核心检查项

| 检查项 | 标准 | 说明 |
|--------|------|------|
| □ 轻资产模式 | 资本开支/净利润 < 20% | FCF≈净利润 |
| □ 高净现金 | 资产负债率 < 15% | 无有息负债负担 |
| □ 品牌护城河 | 定价权/垄断性 | 可持续竞争优势 |
| □ 现金流稳定 | 经营现金流连续3年为正 | 非一次性因素 |

## 5. 类比延伸(Analogical Extension)

**目的**: 当发现一个符合V4.0标准的标的时，系统化寻找同类机会，形成可复用的筛选框架。

### 5.1 核心特征提取(Core Feature Extraction)

从已验证标的中提取关键特征维度：

| 维度 | 当前标的特征 | 量化标准 |
|------|-------------|----------|
| **估值锚定** | 破净幅度 | PB < 0.8 |
| **盈利结构** | 核心利润来源 | 单一子公司贡献 > 50% |
| **现金流质量** | 分红稳定性 | 连续3年分红且派息率 > 30% |
| **业务属性** | 垄断/半垄断 | 区域公用事业或牌照壁垒 |
| **控股结构** | 母公司-子公司架构 | 持股比例 > 50%或实际控制 |

### 5.2 同类标的扫描(Similarity Scanning)

基于核心特征，按优先级扫描潜在标的：

**第一优先级：同构异质**
- 相同控股结构 + 不同行业
- 例：华润医药(医药) → 北京控股(公用事业) → 粤海投资(水务)

**第二优先级：同域异构**
- 相同地域 + 不同业务结构
- 例：天津发展(天津泰达系) → 其他直辖市/开发区控股平台

**第三优先级：同质异价**
- 相同行业 + 估值差异
- 例：内银H股中估值最低的建行H vs 工行H

### 5.3 验证清单(Validation Checklist)

对每个潜在同类标的，强制验证以下项目：

| 检查项 | 验证标准 | 通过标准 |
|--------|----------|----------|
| □ 估值水平 | PB < 1 或 PE < 10 | 符合其一 |
| □ 盈利穿透 | 核心子公司利润可穿透计算 | 能量化D1 |
| □ 分红记录 | 近3年分红连续性 | 无中断 |
| □ 负债安全 | 有息负债/经营现金流 < 3倍 | 财务稳健 |
| □ 控制权 | 母公司对核心子公司实际控制 | 并表或实质控制 |

### 5.4 迭代反馈(Iterative Feedback)

```
发现新标的 → 用V4.0模板验证 → 是否符合金龟/银龟？
     ↓
   符合 → 提取新特征 → 更新核心特征库 → 扩展扫描范围
     ↓
  不符合 → 记录偏差原因 → 优化筛选标准 → 避免同类误判
```

## 6. 最终决策综述(The Verdict)

### 6.1 价格剪刀差分析

对比V2.0与V3.0的价格偏离度：

| 当前价格位置 | 评级 | 操作建议 |
|-------------|------|----------|
| 低于V3.0保底价 | 🐢🍊 **金龟(极度低估)** | 无脑买入 |
| 处于V3.0与V2.0之间 | 🐢💎 **银龟(合理偏低)** | 分批建仓 |
| 高于V2.0价格 | 🐢💰 **铜龟(观察区)** | 仅持有/观望 |

### 6.2 决策流程图

```
开始分析
    ↓
数据清洗(A1/B1/C1/D1/Mal/NetCash)
    ↓
计算V2.0收息买点
    ↓
计算V3.0保底市值
    ↓
计算V3.5 FCEV估值(如适用)
    ↓
当前价格 vs 三个买点
    ↓
  ┌──────────┬──────────┬──────────┐
  ↓          ↓          ↓
低于V3.0   V3.0-V2.0之间  高于V2.0
或FCEV金龟 或FCEV银龟
  ↓          ↓          ↓
金龟🍊      银龟💎      铜龟💰
  ↓          ↓          ↓
无脑买入   分批建仓    观望/持有
```
