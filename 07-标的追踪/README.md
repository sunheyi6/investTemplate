# 港股标的自动追踪系统

> **全自动定时追踪**，无需手动操作，达到目标买点时自动提醒

---

## 🚀 功能特点

- ✅ **全自动运行**：GitHub Actions 每天收盘后自动执行
- ✅ **自动获取数据**：从 Yahoo Finance 获取实时股价
- ✅ **智能提醒**：达到目标买点时自动创建 GitHub Issue 提醒
- ✅ **可视化图表**：自动生成价格走势图
- ✅ **历史记录**：保存所有价格数据，支持回溯分析

## 🔧 手动触发

如果你想立即运行追踪，可以手动触发：

1. 进入 GitHub 仓库

3. 选择 **每日港股标的追踪** 工作流
4. 点击 **Run workflow** 按钮

## ⚙️ 自定义配置

### 修改标的清单
编辑 `auto_stock_tracker.py` 文件中的 `STOCKS` 字典：

```python
STOCKS = {
    '1398.HK': {
        'name': '工商银行',
        'target_drop': 10,  # 目标跌幅
        'category': '内银股',
    },
    # 添加或删除标的...
}
```

### 修改目标跌幅
在 `STOCKS` 配置中调整 `target_drop` 值：
- 保守型：10-15%
- 激进型：5-10%

### 修改运行时间
编辑 `.github/workflows/daily_stock_tracking.yml`：

```yaml
on:
  schedule:
    # 香港时间 16:30 = UTC 08:30
    - cron: '30 8 * * 1-5'
```

## 🔔 提醒机制

### 可买提醒
当某标的跌幅达到目标跌幅时：
1. 在报告中标记为 🔴 **可买**
2. 自动创建 GitHub Issue 提醒
3. Issue 包含详细的价格信息和分析建议

### 观察提醒
当标的接近目标买点时：
1. 在报告中显示距离目标的百分比
2. 可以手动关注，准备分析
