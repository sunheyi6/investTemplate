## 四、分析流程（必须严格执行）

### 步骤0：框架选择（分析前必做）⭐⭐⭐⭐⭐

拿到一个标的，**必须先回答一个问题**：市场在交易这个标的的什么逻辑？

**框架决策树**：
```
拿到一个标的
    ↓
问：近期是否有明确的重大事件催化剂？
├── 有 → 控制权变更/定增入主/资产注入/重组/借壳？
│   ├── 是 → 🔴 事件驱动型 → 用事件驱动框架
│   └── 否 → 特殊事件（诉讼/政策/分红）？→ 也属事件驱动
│
├── 没有 → 问：公司当前处于什么状态？
    ├── 利润大幅下滑/亏损，但行业在触底？ → 🟠 困境反转型
    ├── 利润稳定、股息高、现金流充裕？ → 🟢 纯硬收息型
    ├── 利润波动大、周期性强？ → 🟠 周期拐点型
    ├── 高成长、低利润、零派息？ → ⚫ 成长股型
    ├── 净现金 > 市值、PB < 0.5？ → 🐢🍊 烟蒂股型
    ├── 被ST或面临退市风险？ → ⚫ ST套利型
    └── 以上都不符合 → 🟡 价值回归型 → 用标准价值模板
```

**你的判定**：请明确写出该标的是哪种框架类型，以及为什么。

### 步骤3：估值硬约束

**PE 计算（必须同时展示）**：
```
静态PE = 市值 / 最近完整年度净利润
TTM PE = 市值 / 最近四个季度净利润之和
两者差异及原因：
硬约束以 TTM PE 优先（更能反映最新盈利趋势）
例外：若TTM失真（如强季节性行业），以年度PE为准
```

**估值纪律四条（V5.5.20）**：
1. **表面估值优先**：市场按表面估值交易，不以调整后估值自我安慰
2. **表面PE<10倍硬约束**（价值回归型/纯硬收息型）
3. **净现金占比>20%-30%但不抵扣估值**（避免净现金估值陷阱）
4. **股息率>4%-5%**（绑定表面PE）

## 六、重要纪律

1. **数据源等级**：S级（年报原文）> A级（业绩公告）> B级（第三方平台）> C级（媒体）> D级（AI估算）。AI估算**禁止用于投资决策**。
2. **港股股本结构**：市值 = 股价 × 总股本（不是H股流通股本）。
3. **净现金陷阱**：净现金>市值时不能自动判定为烟蒂股，必须验证现金真实性、业务可持续性。
4. **不做大学生数学题**：只做简单确定的"小学生数学题"，回避复杂预测。
5. **反对压票思维**：当估值达公允值，因机会成本果断卖出切换。
