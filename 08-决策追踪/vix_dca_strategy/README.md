## 策略核心逻辑

**恐慌时贪婪，乐观时谨慎**

- **高VIX（恐慌）** → 定投日多买
- **低VIX（乐观）** → 定投日减仓
- **执行频率** → 每两周的周二执行买卖
- **收益追踪** → 每日更新收益率和盈亏

## 常见问题

**Q: 为什么收益每天更新，但买卖只在定投日？**  
A: 为了追踪每日收益变化，但避免频繁交易和手续费损耗。

**Q: 非定投日VIX突破30，可以额外买入吗？**  
A: 不建议。策略设计就是双周定投，避免情绪化交易。

**Q: 每日收益如何计算？**  
A: 当日收盘价 × 持仓份额 - 投入成本 = 当日浮动盈亏。
