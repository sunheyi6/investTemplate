## 项目定位

本项目是一个**港股/A股个股投资分析模板系统**，属于知识管理型项目，非传统软件工程。核心产出是一套标准化的投资分析框架和配套工具，帮助投资者进行深度个股研究。

| 属性 | 说明 |
|------|------|
| **类型** | 投资分析模板 + 辅助工具脚本 |
| **目标用户** | 个人投资者（当前资金规模：十来万港币） |
| **投资市场** | 港股（优先）、A股 |
| **投资流派** | 四流派体系（纯硬收息/价值发现/烟蒂股/关联方资源型） |

## 投资哲学核心（必读）⭐⭐⭐⭐⭐

### 1. 龟龟投资理论（防御型价值投资）
- **防御性强**：市场下跌时回撤小
- **追求绝对防御能力**：牛市80%时间跑赢，熊市90%以上时间跑赢
- **收益结合回撤看**：注重赔率与胜率的综合预期值

### 2. 普通人假设（能力圈认知）
- 视线仅限**1-2年**，无信息优势
- 回避预测大宗商品等高波动变量
